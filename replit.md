# Saqlain Hub

A simple static personal hub website by Saqlain Ejaz.

## Project Structure

- `index.html` - Main HTML page with navigation and clock container
- `style.css` - Styles for the navbar, clock, and layout
- `script.js` - Digital clock and date display logic

## Features

- Responsive navigation bar
- Live digital clock (12-hour format with AM/PM)
- Date display

## Running the App

The site is served as static files using `serve` on port 5000.

**Workflow:** `Start application` — runs `npx serve . -l 5000`

## Deployment

Configured as a **static** deployment with `publicDir: "."`.
